# OceanGoal — Versiones del proyecto (1.0 y 2.0)

Este repositorio contiene dos versiones del proyecto organizadas por carpetas:

- `OceanGoal1.0/`
- `OceanGoal2.0/`

## Objetivo
Mantener y comparar iteraciones del proyecto (v1 vs v2), permitiendo revisar cambios y evolución entre versiones.

## Cómo usar
1. Entra a la versión que quieras revisar (`OceanGoal1.0` o `OceanGoal2.0`).
2. Ejecuta el proyecto según la tecnología usada dentro de esa carpeta.

## Nota
Si dentro de cada versión hay un stack específico (por ejemplo, web, móvil, Python, etc.), puedo ajustar este README con instrucciones exactas de instalación y ejecución.
